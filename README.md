# Un-Fisheye-Image
Convert distorted lens effect back into straight image

# Install
requirement: opencv and numpy

# Usage
Insert opencv image to code

FOV = slider for real camera fov
PFOV = slider for converting fisheye pov to perspective FOV
PAD = padding (unused)
DISTORTION = 0 = linear
            1 = equalarea
            2 = orthographic
            3 = stereographic
